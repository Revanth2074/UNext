# Student Course Registration System


## Project Overview

The Student Course Registration System is a Python-based application developed using Object-Oriented Programming concepts.

The system manages students, courses, and enrollment operations.


## Features

### Course Management

- Add new courses
- View available courses
- Track seat availability
- Store course data in CSV files


### Student Management

- Register students
- Store student details
- Track enrolled courses


### Enrollment Operations

- Enroll students into courses
- Drop registered courses
- Prevent enrollment when seats are full
